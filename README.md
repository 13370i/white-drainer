## 🚀 Деплой (варианты)

### Вариант 1: Vercel (бесплатно, быстро)
```bash
# Установи Vercel CLI
npm i -g vercel

# Задеплой
cd white-drainer-landing
vercel --prod
```

### Вариант 2: Netlify Drop
1. Зайди на https://app.netlify.com/drop
2. Перетащи папку `white-drainer-landing`
3. Получишь URL instantly

### Вариант 3: GitHub Pages
1. Создай репо `white-drainer`
2. Запушь файлы
3. Включи GitHub Pages в settings

### Вариант 4: Base Mini App
- Гайд: https://docs.base.org/mini-apps/quickstart/create-new-miniapp
- Можно сделать как mini app для Base экосистемы

## ⚠️ Security Notes

**Важно:** Лендинг НЕ собирает приватные ключи напрямую.

**Текущий подход:**
- Кнопки ведут на Telegram (@LEET0i)
- Клиент пишет напрямую в зашифрованный канал

**Рекомендации для клиентов:**
1. Использовать ProtonMail/Signal для передачи ключей
2. Шифровать ключи паролем перед отправкой
3. Использовать PrivateBin для одноразовых ссылок

**Если нужна форма на сайте (будущая версия):**
- Client-side encryption перед отправкой
- HTTPS обязательно
- No database storage (in-memory only)
- Auto-delete после обработки
